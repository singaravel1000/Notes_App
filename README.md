## How to Use

1. Open `index.html` in your web browser
2. Type a title and content for your note
3. Click **"Add Note"** to save
4. Use the **Search** bar to find notes quickly
5. Click **"Edit"** to modify or **"Delete"** to remove notes

## Storage

All notes are stored locally in your browser using **LocalStorage**, so they persist even after closing the browser.
